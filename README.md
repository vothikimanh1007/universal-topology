## 🚀 Getting Started

### 1\. Running the 3D Web Interface

The web interface is a single, zero-dependency HTML file.

- **Local:** Simply double-click index.html to open it in any modern browser.
- **Deployment:** You can drag and drop the repository into [Netlify Drop](https://app.netlify.com/drop) or host it directly via **GitHub Pages**.

### 2\. Running the Python Pipeline

The Python script (Knowledge_Systems_Topology.py) processes the raw PDFs, generates the semantic networks, runs Latent Dirichlet Allocation (LDA), builds the TDA Mapper projections, and generates all academic figures (PNGs) and datasets (CSVs).

**Prerequisites:**

pip install networkx matplotlib PyPDF2 kmapper scikit-learn wordcloud nltk numpy

**Execution:**

python Knowledge_Systems_Topology.py

_Note: The script will automatically download the necessary NLTK corpora (punkt, stopwords, vader_lexicon) on its first run._

## 🔬 Features & Methodology

- **Advanced N-Gram Ontology Mapping:** Detects cross-cultural theological multi-word expressions (e.g., _"dependent origination"_, _"son of man"_) and normalizes them into a shared vector space.
- **Ego-Network Decomposition:** Solves the classic network "hairball" problem by extracting Radius-1 ego-networks for deep qualitative analysis of each Super-Node.
- **Explainable AI (XAI) Trajectory Tracing:** Allows users to input an arbitrary, multi-tradition sentence and visually trace the algorithm's semantic routing through the global topology.
- **Topological Data Analysis (TDA):** Utilizes KeplerMapper (DBSCAN + PCA) to project the high-dimensional linguistic space into continuous simplicial complexes, proving the structural isomorphism of human spirituality.
- **Real-World Applications:** Framework includes translations of these models into practical tools for **AI Alignment (Ethics)**, **Cognitive Behavioral Therapy (CBT)**, and **Organizational Design**.
